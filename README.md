# Password Manager

This is a simple password manager application built using Node.js, Express.js, HTML, CSS, and JavaScript. It allows users to securely store their passwords with a friendly interface.

## Features

- Add and manage passwords.
- Toggle visibility of passwords.
- Delete passwords when no longer needed.
- Responsive design for better accessibility.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: In-memory storage (using an array)

## Getting Started

1. Clone the repository:
   ```bash
   git clone sreyas-cheviri/passmanager.git
   ```
2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
5. Open your browser and go to `http://localhost:3000`.

## Usage

- Enter a name and password in the input fields.
- Click the "+" button to add the password to the list.
- Click on the password entry to strike it out (indicating deletion).
- Use the toggle button to show/hide password visibility.
- Click the down button to show/hide the password list.
